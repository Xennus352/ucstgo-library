# 📚 UCSTGO Digital University Library Knowledge Base

---

## 🏛️ Library Rules

- စာအုပ်ငှားကာလမှာ 14 ရက် (2 weeks) ဖြစ်သည်
- Due date မတိုင်မီ ပြန်မအပ်ပါက borrowing restriction သို့မဟုတ် penalty ရှိနိုင်သည်
- စာအုပ်ပျက်စီးခြင်း သို့မဟုတ် ပျောက်ဆုံးခြင်းဖြစ်ပါက တာဝန်ယူရမည်
- Library property ကို လေးစားစွာ အသုံးပြုရမည်
- အခြားသူများ၏ စာအုပ်များကို ခွင့်မပြုဘဲ မယူရ

---

## 📖 Ebook Rules

- Ebook များကို registered university students / members သာ အသုံးပြုနိုင်သည်
- Ebook download ပြုလုပ်ခြင်းကို system policy အရ ခွင့်မပြုနိုင်သည်
- Ebook files များကို share / redistribute / sell မပြုလုပ်ရ
- Academic use အတွက်သာ အသုံးပြုရန် ရည်ရွယ်ထားသည်
- Login credentials များကို အခြားသူများနှင့် မမျှဝေရ

---

## 📥 Borrowing Guide

1. Library system ထဲတွင် စာအုပ်ကို ရှာဖွေပါ
2. လိုချင်သော စာအုပ်ကို ရွေးပြီး Borrow ခလုတ်ကို နှိပ်ပါ
3. Librarian သို့မဟုတ် system approval စောင့်ပါ
4. Approval ရပြီးနောက် စာအုပ်ကို access လုပ်နိုင်မည်
5. Due date မတိုင်မီ ပြန်အပ်ရန် အရေးကြီးသည်

---

## 🔁 Returning Rules

- Due date မတိုင်မီ စာအုပ်ကို ပြန်အပ်ရမည်
- Late return ဖြစ်ပါက borrowing restriction သို့မဟုတ် penalty ရှိနိုင်သည်
- Damaged book ဖြစ်ပါက replacement cost ပေးရန်လိုအပ်နိုင်သည်

## 📌 Library Ethics & Conduct

- Library system ကို လိမ်လည်အသုံးပြုခြင်း မပြုရ
- Other users ကို disturb မလုပ်ရ
- Academic integrity ကို လေးစားရမည်
- System security ကို ချိုးဖောက်ခြင်း မပြုရ
- University policies အားလုံးကို လိုက်နာရမည်
